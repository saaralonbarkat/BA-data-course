Gapminder Modern Dataset (1952–2022)

Exploring global trends in population, life expectancy, GDP, and CO₂

Published

April 21, 2026

Overview

This report explores the Gapminder Modern dataset, which combines four indicators for up to 195 countries across 15 five-year snapshots from 1952 to 2022:

Indicator Description
lifeExp Life expectancy at birth (years)
pop Total population
gdpPercap GDP per capita (inflation-adjusted USD)
co2Percap CO₂ consumption per capita (tonnes)
Code 
library(tidyverse)
library(scales)

gap <- read_csv("gapminder_modern_1952_2022.csv") |>
  filter(!is.na(continent))   # drop the 1 unmatched row

continent_colours <- c(
  Africa   = "#E69F00",
  Americas = "#56B4E9",
  Asia     = "#009E73",
  Europe   = "#CC79A7",
  Oceania  = "#F0E442"
)

Life Expectancy

Global Trend

Life expectancy has risen dramatically in every continent. The shaded ribbon shows the min–max country range; the line shows the population-weighted mean.

Code 
gap |>
  filter(!is.na(lifeExp), !is.na(pop)) |>
  summarise(
    mean_le  = weighted.mean(lifeExp, pop),
    min_le   = min(lifeExp),
    max_le   = max(lifeExp),
    .by = c(continent, year)
  ) |>
  ggplot(aes(year, mean_le, colour = continent, fill = continent)) +
  geom_ribbon(aes(ymin = min_le, ymax = max_le), alpha = 0.12, colour = NA) +
  geom_line(linewidth = 1) +
  scale_colour_manual(values = continent_colours) +
  scale_fill_manual(values = continent_colours) +
  scale_x_continuous(breaks = seq(1952, 2022, 10)) +
  labs(x = NULL, y = "Life expectancy (years)",
       colour = NULL, fill = NULL)

Top & Bottom 10 Countries (2022)

Code 
le_2022 <- gap |>
  filter(year == 2022, !is.na(lifeExp)) |>
  select(country, continent, lifeExp) |>
  arrange(desc(lifeExp))

bind_rows(
  slice_head(le_2022, n = 10) |> mutate(rank = "Top 10"),
  slice_tail(le_2022, n = 10) |> mutate(rank = "Bottom 10")
) |>
  select(rank, country, continent, `Life expectancy` = lifeExp)

The Gapminder View: Wealth vs. Health

A bubble chart replicating the classic Gapminder “wealth vs. health” view for selected years.

Code

         !is.na(gdpPercap), !is.na(lifeExp), !is.na(pop)) |>
  ggplot(aes(gdpPercap, lifeExp,

Summary Statistics (2022)

Code 
gap |>
  filter(year == 2022) |>
  summarise(
    across(c(lifeExp, gdpPercap, co2Percap),
           list(median = \(x) median(x, na.rm = TRUE),
                mean   = \(x) mean(x, na.rm = TRUE),
                sd     = \(x) sd(x, na.rm = TRUE)),
           .names = "{.col}__{.fn}"),
    .by = continent
  ) |>
  pivot_longer(-continent, names_to = c("Indicator", "Stat"),
               names_sep = "__") |>
  pivot_wider(names_from = Stat, values_from = value) |>
  arrange(Indicator, continent) |>
  mutate(across(where(is.numeric), \(x) round(x, 1)))

Data Notes

  • Source: Gapminder (indicators: lex, pop, gdp_pcap, co2_pcap_cons).
  • Years: every 5 years from 1952 to 2022 (15 snapshots).
  • One unmatched ISO3 code with no country name was excluded (~15 rows).
  • Small states (Liechtenstein, Monaco, San Marino) have partial data for some indicators.
  • CO₂ values represent consumption-based emissions per capita.
